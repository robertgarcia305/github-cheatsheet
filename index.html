Github / Git Commands

When wanting to push large files like images, it might throw an error and skip the commit, so abstain from pushing large files, do it step by step

Initializing a repo

## from local folder, how to create repo on github

## you need to go on github and create a new repo

## Initialize a new Git repository

git init

## Add all the files in your project to the repository

git add .

## Create the initial commit

git commit -m "Initial commit"

then

git remote add origin https://github.com/your-username/your-repo-name.git

## If it is the first commit you do:

git push -u origin main

## or it can also be origin master

Troubleshooting the commit

If it doesn't work you do:

git push origin master --force

then git push after that

When there's a repo already on github and want to clone it

you need to go to the repo and copy the link on the green code button, then do the following commands

cd where you want to have the folder

git clone {link you just copied}

cd {repo name}

## to check the status of the repo

git status

git add .

git commit "new commit"

git push

React Project

when installing a new react project it is going to install some a folder called node_modules which dont need to go on github

you need to create a .gitignore file

1) change to the desired directory

touch .gitignore

## on the .gitignore file just add: node_modules/

## or any other files and folders you would like for github to ignore

// if you already had node_modules files to github you run this command:

git rm -r --cached node_modules

git add .gitignore

git commit -m "Ignore node_modules folder"

git push

To push a github repo

## To push a React app created with npx create-react-app to a GitHub repository, follow these steps:

Create a GitHub Repository:

Click the "+" icon in the top right corner and select "New repository."

Follow the prompts to set up your repository. You can choose a name, add a description, set it to public or private, and select other options.

Please don't add a readme.md file, that will be added with your react app

## Navigate to your React app's root directory using the cd command. For example:

## cd path/to/your/my-tests

git init

git remote add origin https://github.com/your-username/your-repo-name.git

git branch -M main

git add .

git commit -m "Initial commit"

git push -u origin main
